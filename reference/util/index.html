    ebcc.util.compress_axes(subscript, array, include_diagonal=False, out=None)

    Compress an array into lower-triangular representations using an einsum-like input.

    Parameters:

    Name Type Description Default
    subscript str

    Einsum-like subscript, where repeated characters are considered part of the same set of indices to be compressed.

    		 required
    array numpy.ndarray

    Array to compress.

    		 required
    include_diagonal bool, optional

    If True, include the diagonal elements in the compression. Default value is False.

    		 False

    Returns:

    Name Type Description
    array_flat numpy.ndarray

    Compressed array, the number of dimensions will be equal to the number of unique characters in subscript, and the size of each dimension will correspond to the output of ntril_ndim for that particular set of indices.

    Examples:

    >>> t2 = np.zeros((4, 4, 10, 10))
>>> compress_axes("iiaa", t2).shape
(6, 45)

    ebcc.util.decompress_axes(subscript, array_flat, shape=None, include_diagonal=False, symmetry=None, out=None)

    Reverse operation of compress_axes, subscript input is the same. The input symmetry is a string of the same length as subscript, with a "+" indicating symmetry and "-" antisymmetry.

    Parameters:

    Name Type Description Default
    subscript str

    Einsum-like subscript, where repeated characters are considered part of the same set of indices to be compressed.

    		 required
    array_flat numpy.ndarray

    Array to decompress.

    		 required
    shape tuple of int

    Shape of the output array.

    		 None
    include_diagonal bool, optional

    If True, include the diagonal elements in the compression. Default value is False.

    		 False
    symmetry str, optional

    Symmetry of the decompression. For each index in subscript, a + indcates symmetry and - indicates antisymmetry. The value of symmetry must be the same for repeated characters in the corresponding subscript.

    		 None

    Returns:

    Name Type Description
    array numpy.ndarray

    Decompressed array.

    Examples:

    >>> t2 = np.zeros((6, 45))
>>> decompress_axes("iiaa", t2, symmetry="----").shape
(4, 4, 10, 10)

    Dispatch an einsum. If symmetry, then assume that all arrays are totally symmetric within occupied, virtual and bosonic sectors. If symmetry is an iterable, then it should provide permutations corresponding to each input array which give the desired symmetry.
